# Linear Search in C

## Requirements

- [gcc](https://gcc.gnu.org/)
- or [clang](https://clang.llvm.org/)
- [make](https://www.gnu.org/software/make/)
- (Optional) [gdb](https://sourceware.org/gdb/)
- (Optional) [lldb](https://lldb.llvm.org/)

## Run Tests

```bash
make test
```

### Run Tests in Debugger

```bash
make test-debug
```
