# Binary Search in JavaScript

[![js](https://github.com/claudemuller/algorithms/actions/workflows/binary-search.js.yml/badge.svg)](https://github.com/claudemuller/algorithms/actions/workflows/binary-search.js.yml)

The [companion article](https://dxt.rs/category/programming/general/binary-search) discussing binary search can be found here.

## Requirements

- [node](https://nodejs.org/en)
- [yarn](https://yarnpkg.com/)
- or [npm](https://www.npmjs.com/)

## Install Project Dependencies

```bash
yarn

// Or..
npm install
```

## Run Tests

```bash
yarn test

// Or..
npm test
```
